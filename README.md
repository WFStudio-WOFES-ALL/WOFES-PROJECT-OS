Вот готовый текст для `README.md` в # WOFES OS

Сборка минимальной операционной системы на базе Debian с графическим окружением (Firefox). Автоматическая сборка через GitHub Actions.

# Особенности

- База: Debian stable
- Ядро: Linux (последнее стабильное)
- Графика: Firefox (можно заменить на WOFES SFF)
- Live-система: работает с флешки без установки
- Сборка: полностью автоматическая, через GitHub Actions

# Как собрать

1. Форкни или клонируй репозиторий
2. Добавь свои файлы (например, папку с браузером WOFES SFF)
3. GitHub Actions сам соберёт ISO
4. Скачай артефакт `wofe-os.iso` из вкладки Actions

# Как запустить

- В эмуляторе: `qemu-system-x86_64 -cdrom wofes-os.iso`
- На флешку: `dd if=wofes-os.iso of=/dev/sdX bs=4M`
- На реальном железе: записать через Rufus (Windows) или BalenaEtcher

# Структура репозитория

```
WOFES-PROJECT-OS/
├── .github/
│   └── workflows/
```
